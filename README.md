Universidad de Burgos

Descripción

Este repositorio contiene la documentación y los archivos fuente desarrollados para el Trabajo Fin de Grado titulado "Diseño y desarrollo de un dispositivo ponible de seguridad para la detección de proximidad en líneas de tensión de 15, 20 y 30 kV".

El proyecto consiste en el diseño, implementación y validación de un dispositivo electrónico ponible orientado a la detección de proximidad a líneas eléctricas de media tensión mediante la medida del campo eléctrico ambiental, con el objetivo de mejorar la seguridad de los trabajadores.

Contenido del repositorio:
Análisis y simulaciones en MATLAB/Simulink.
Firmware desarrollado para STM32.
Diseño electrónico en KiCad.
Diseño mecánico de la envolvente en AutoCAD.
Planos del proyecto.
Hojas de datos de los principales componentes utilizados.

Algunos archivos y esquemas conservan la denominación de desarrollo "Detector de Campo Eléctrico – EPI Wearable", utilizada durante las fases iniciales del proyecto. Dicha denominación hace referencia al mismo dispositivo desarrollado en este Trabajo Fin de Grado.

La documentación técnica incluida en este repositorio se encuentra protegida mediante contraseña. Facilitaré la clave de acceso previa solicitud razonada, estableciendo un canal de comunicación directa con investigadores, docentes o profesionales interesados en el proyecto. De este modo, se pretende asegurar que cualquier reutilización del hardware, firmware o documentación técnica se realice con el contexto adecuado, comprendiendo tanto las limitaciones del prototipo como las consideraciones técnicas necesarias para su correcta interpretación.
